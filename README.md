# Fitrah Wahyudi — Personal Web Portfolio

A personal portfolio website for **Fitrah Wahyudi**, built with [Next.js](https://nextjs.org) and [Tailwind CSS](https://tailwindcss.com).

🔗 LinkedIn: [https://www.linkedin.com/in/fitwahyudii/](https://www.linkedin.com/in/fitwahyudii/)

## Features

- **Hero** — Introduction with name, tagline, and LinkedIn/GitHub links
- **About** — Personal bio section
- **Skills** — Grouped technology cards (Frontend, Backend, Tools, Other)
- **Projects** — Showcase of notable projects with tech stack tags
- **Contact** — Contact links and call-to-action
- **Responsive** — Mobile-first design with a hamburger menu

## Tech Stack

- [Next.js 15](https://nextjs.org) (App Router)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS v4](https://tailwindcss.com)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Build

```bash
npm run build
npm start
```
