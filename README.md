# Portfolio Web

A modern, minimalist portfolio for **Wahyudi** - Software Developer & Systems Architect.

## Features

- **Hero Section** - Punchy value proposition with AI Mastery focus
- **Case Studies** - Architectural diagrams for Fintech, Mobility & Utility platforms
- **Tech Stack** - Clean categorized grid (Go, Python, Laravel, Docker, etc.)
- **Dark Mode** - Tech-forward design with JetBrains Mono typography
- **Responsive** - Mobile-first approach

## Tech Stack

- HTML5 / CSS3 (Vanilla)
- Node.js static file server
- Docker ready

## Quick Start

### Local Development

```bash
npm install
npm start
```

Visit `http://localhost:3000`

### Docker

```bash
docker build -t portfolio .
docker run -d -p 3000:3000 portfolio
```
